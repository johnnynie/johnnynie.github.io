css基本选择器

css css css选择器 2019/12/02

一、css基本选择器列表：

选择器名称 选择内容 示例
元素选择器（也称作标签或类型选择器） 所有指定类型的 HTML 元素 h1 { }
通用选择器 所有类型的 HTML 元素 * { }
类选择器 具有特定类的元素（单一页面中，一个类可以有多个实例） .box { }
ID 选择器 具有特定 ID 的元素（单一 HTML 页面中，每个 ID 只对应一个元素，一个元素只对应一个 ID） #unique { }
属性选择器 拥有特定属性的元素 a[title] { }
伪类选择器 特定状态下的特定元素 p:first-child { }(向元素的第一个子元素添加样式。)，a:hover{ }
伪元素选择器 用于向某些选择器设置特殊效果 p::first-line { }
后代选择器 后代HTML元素选择器 article p
子选择器 直接子HTML元素选择器 article > p
相邻同胞选择器 相邻同胞HTML元素 h1 + p
同胞选择器 同胞HTML元素 h1 ~ p

二、特殊说明

1、伪类选择器

常见伪类选择器包含：

  • 锚伪类a:link（未访问的链接）、a:visited（已访问的链接）、a:hover（鼠标移动到链接上）、a:active（选定的链接）
  • :first-child，选择元素的第一个子元素

2、伪元素选择器

按照规范，伪元素选择器应该使用双冒号（::）而不是单个冒号（:），以便区分伪类和伪元素。但是，由于旧版本的 W3C 规范并未对此进行特别区分，因此目前绝大多数的浏览器都同时支持使用这两种方式来表示伪元素。

常见的伪元素选择器包含：

  • ::first-line，向文本的首行设置样式（第二行将不会应用样式）
  • ::first-letter，向文本的首字母设置特殊样式（其他字母不会应用样式）
  • ::before，可以在元素的内容前面插入新内容
  • ::after，可以在元素的内容之后插入新内容
