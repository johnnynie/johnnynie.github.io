前端开发基础知识速查网站推荐

前端工具 前端参考手册 2019/10/18 
本篇文章推荐一个前端开发基础知识速查网站

学习前端，离不开前端的基础知识，包含html、css、javascript等。这些基础知识比较多，无论我们处于哪个级别的前端开发人员，都有可能会需要去查询某个知识点的时候。以下就是推荐的查阅网站：MDN web docs

MDN 的一切（文档和网站本身）都是由一个开放的开发者社区创造的。这个网站涵盖了html、csss、javascript等的基础知识，且保持较快的技术知识点更新（这点很重要，因为技术点是在不断发展的）且比较权威，我们可以把这个作为一个参考手册，有需要的时候就到这里查一下。
